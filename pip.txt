pypi 镜像使用帮助

临时使用
    pip install -i https://pypi.tuna.tsinghua.edu.cn/simple some-package

设为默认
    python -m pip install --user --upgrade pip  #更新pip
    pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple


Windows
1.安装虚拟环境python包：pip install virtualenv

2.到想要创建虚拟环境的路径下，运行cmd，输入virtualenv envs  (envs为自定义名称）
	 本目录下会生成一个envs文件夹

3.进入虚拟环境
   	打开envs文件夹，再打开Scripts文件夹 cmd 运行  activate

4.安装
    pip install opencv-python

    pip install reqiure.txt

5.pip查看已安装的包
    pip show packagename

6.列出所有的安装包
    命令：pip list
 
7.pip检测更新
    pip list --outdated

8.pip升级包
    pip install --upgrade packagename

9.pip卸载包
    pip uninstall packagename

10.安装requirements.txt中的包
    pip install -r requirements.txt

11.更新数据
    python -m pip install --upgrade pip

Linux
python虚拟环境：允许安装不同版本的套件

创建：mkvirtualenv  虚拟环境名称                 
进入：workon  虚拟环境名称
查看所虚拟环境: workon  + 回车
删除：1)先退出虚拟环境   2） rmvirtualenv 虚拟环境名称
退出：deactivate
创建虚拟环境，指定Python路径： mkvirtualenv -p   /usr/bin/python3.6  xxx
在虚拟环境中安装套件： pip install 套件名==版本

问题: 
    Ignoring invalid distribution -ip
    删除对应目录中以 ~ip 开头的目录即可
